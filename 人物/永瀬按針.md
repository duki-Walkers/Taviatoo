# 永瀬按針（ながせ あんじん）

本名：永瀬按針。通称：William Nagase Howard（按針＝William Adams）。1990年代にロディオンでOld Mr. Howardと接触・薫陶を受けた最後の世代かもしれない。星川書房南又店副店長・昼は泊灯書房を経営。

第1話に「店員は書架の整理をしている。割と夢中そうだ。」として既に登場。中高時代の春・森本の学校へ教科書搬入で来ており、春と接触した。

**不死身の可能性——作品の全時間軸を静かに貫いている。誰も触れない。**

凪が泊灯書房と南又店を同じ日に訪れた際「泊灯と同じ人 name 永瀬 なぜ 両方いる？」とメモした。答えは出なかった。
