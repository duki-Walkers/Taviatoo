# 秋山凪（あきやま なぎ）

都市エスノグラフィー系研究者。富智央市育ち。馬律大→東央大真壁研院進。郊外や滞留空間を歩き続け、都市の痕跡を観察する。速記を習得。Y+23年1月の合同ゼミで夕と出会い、ロディオンへ留学。留学後は台北・ベルリン・ソウルを漂流。Y+33年帰国。享年67。**終生の住処は南又。**

---

## 研究の軌跡

```
卒論——都市における滞留空間と公共性
　　　ジェイコブス的な「目」×ハーバーマス的な公共圏論
　　　×南又・内海谷・富智央での観察
修士以降——フィールドワーク先が広がる
　　　　　（夕と絡み始めてから出会う人々）
```

---

## 後年の著作より

「滞留する場所には、人が去った後も何かが残る。それは記憶でも記録でもなく、ただそこにある空気のようなものだ。名前のつけられない何かが、痕として場所に染み込んでいる。」

## 学生生活

**馬律大カリキュラム**
```
1年次——1年次ゼミ（ランダム振り分け）→佐伯ゼミ
2年次——社会調査実習・量的質的調査法
3年次——ゼミ（必修）→佐伯ゼミ継続
4年次——ゼミ継続・卒論
```

**1年次（Y+19年）**
南又の部屋——六畳弱・窓から又川の方向に空。最初のゼミの日にメモ帳を見られた。それだけだった。新歓焼肉で春と「またいる。」「またか。」

**2年次（Y+20年）**
社会調査実習——南又周辺の郊外空間を対象。佐野美樹と出会う。陽（16歳）と又川河川敷で初会い。

**3〜4年次（Y+21〜23年1月まで）**
佐伯ゼミ本格化・卒論テーマの絞り込み・SNS開始・院試（佐伯経由で真壁研を知る）

## 凪の部屋

六畳弱。窓を開けると又川の方向に空が広がっていた。「良い景色だった。」本とメモ帳が大半の荷物だった。
