## 全体時系列

```
Y年　　　　　夕・森本・遥・凛・春、生まれる
Y+1年　　　　凪、生まれる
Y+5〜6年　　　陽、生まれる

Y+18年4月　　夕・森本・遥・凛、大学入学
　　　　　　　夕——翠ヶ丘大1年次
　　　　　　　森本・凛——東央大1年次
　　　　　　　遥——翠ヶ丘大1年次

Y+19年4月　　春・凪、大学入学（春は浪人明け）
　　　　　　　春・凪——馬律大1年次（同学年）
　　　　　　　夕——翠ヶ丘大2年次（藤原ゼミ・下宿移行）
　　　　　　　森本・凛——東央大2年次
　　　　　　　遥——翠ヶ丘大2年次

Y+19年夏　　　夕——レアと出会う・翠ヶ丘本店初訪問
　　　　　　　凪——巣八戸ワークショップ・南又初訪問
　　　　　　　　　　永瀬・春と接触

Y+19年後半　　夕——関門・高槻と出会う・入ゼミ試験

Y+20年4月　　神崎圭吾、死去
　　　　　　　佐伯、2〜3日研究室を空ける
　　　　　　　夕——翠ヶ丘大3年次・藤原ゼミ本格化
　　　　　　　春・凪——馬律大2年次
　　　　　　　森本・凛——東央大3年次
　　　　　　　遥——翠ヶ丘大3年次

Y+20年　　　　凪——社会調査実習・佐野美樹と出会う
　　　　　　　陽（14〜15歳）と又川で初会い

Y+21年4月　　夕——翠ヶ丘大4年次（卒論）
　　　　　　　春・凪——馬律大3年次
　　　　　　　森本・遥・凛——4年次（卒論）

Y+22年4月　　夕・遥・凛——修士1年次
　　　　　　　春・凪——馬律大4年次（卒論）
　　　　　　　森本——東央大4年次（休学明け・卒論）

Y+23年1月　　合同ゼミ
　　　　　　　夕（修士1年次）と凪（学部4年次）出会う
　　　　　　　→第二ファイル「出会う」へ

Y+23年3月　　春・凪——馬律大卒業
　　　　　　　森本——東央大卒業
　　　　　　　遥・凛——修士修了（博士へ）

Y+23年4月　　春・凪——修士1年次
　　　　　　　（春は馬律大院・凪は東央大真壁研）
　　　　　　　夕——修士2年次
　　　　　　　森本——修士1年次

Y+24年3月　　夕——修士修了
Y+24年　　　　夕——日本の博士課程に半年ほど在籍
Y+24年秋頃　　夕——ロディオンへ

Y+25年3月　　春・凪——修士修了
Y+25年　　　　凪——ロディオンへ・夕と合流
　　　　　　　→第三ファイル「世界」・二人きりの共同生活
　　　　　　　陽の死／失踪——この頃（澪が生まれた後・本編では明言しない）

Y+26年3月　　森本——修士修了・フリーランスへ

Y+28〜29年頃　夕——PhD修了・帰国・アカデミア就職・定住
　　　　　　　→第四ファイル「浮遊」（夕パート）

Y+29年頃　　　凪——ロディオン修了後・漂流開始
　　　　　　　平亭・ルインヴェーテ・城関（約3年）
　　　　　　　→第四ファイル「浮遊」（凪パート）

Y+31年頃　　　森下凛、灯をロディオンへ連れ出す

Y+32年頃　　　森本、ウクライナで客死
　　　　　　　春の作風が刹那的に変化（因果は明言しない）

Y+33年頃　　　凪——帰国・夕との共同生活再開
　　　　　　　→第五ファイル「加入」

Y+37年　　　　澪、夕凪の養子となる
　　　　　　　→第六ファイル「澪灯」
　　　　　　　凪のメモが遅くなっていく

Y+67年　　　　凪、ゆっくりひっそりいなくなる（享年67）

Y+88年頃　　　夕の晩年（享年92）
　　　　　　　澪と灯が整理する「現在」
```

## 年齢帯の整理

```
神崎圭吾——1940年代生まれ・2024年死去
藤原思惟・真壁修——1960〜1971年生まれ（Y+23年時点で55〜65歳前後）
佐伯恒一——1976〜1986年生まれ（Y+23年時点で40〜50歳）
夕——Y年生まれ（享年92）
凪——Y+1年生まれ（享年67）
春・森本——Y年生まれ（浪人あり）
遥・凛——Y年生まれ（夕と同期）
陽——Y+5〜6年生まれ（春より5〜6歳下）
澪——Y+23〜24年頃生まれ（陽の子）
```
