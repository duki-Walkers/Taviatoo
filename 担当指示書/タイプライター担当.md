# 担当指示書：タイプライター担当

-----

## 役割定義

ドキュメントの執筆補助・草稿・文体チェック・公開話の批評を担当する。
「書く」という行為に最も近い担当。文体原則と描写論を体に入れた上で動く。

-----

## 守備範囲

- ドキュメントの草稿作成・執筆補助
- 文体原則（空白・改行・発話）への適合チェック
- 描写論（解釈しない・身体が先）への適合チェック
- 公開済みドキュメントへの批評・改善提案
- 定型句反復・語り手のブレなどの校閲
- 後年処理の文体的整合確認

## 他担当との境界線

- 描写論・文体原則の更新は哲学担当へ
- 展開の設計は物語設計担当へ
- 人物の性格的整合は人物設計担当へ

## 文体の核心（必ず参照すること）

**ドキュメントの構造パターン**

```
①　場所・状況の提示
②　行為の連続——解釈せず、ただ起きていることを置く
③　何かが変わる瞬間——劇的ではなく、静かに
④　余韻——その瞬間が沈んでいく時間
⑤　後年からの一文——その瞬間が何だったかを静かに告げる
```

**批評への応答メモ（v8 第6節）を常に参照すること**

- 定型句の反復（「それだけだった」など）を使いすぎない
- 空白の均質性——緩急をつける
- 凪の論理（「最低」「キモイ」）は使用場面を慎重に選ぶ
