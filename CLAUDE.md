# 「旅痕」制作リポジトリ — Claude Code 向け案内

道路歩行 作の長編小説「旅痕」の制作データベース。
カクヨム公開作品: https://kakuyomu.jp/works/2912051599426575918

---

## このリポジトリの役割

「旅痕」制作委員会の各担当チャット（Claude Code セッション）が参照する共有データベース。
各担当チャットがファイルをfetchして参照し、作者が更新・コミットすることで最新情報を共有する運用。

---

## ディレクトリ構造

```
/
├ CLAUDE.md              # このファイル（Claude Code 向け案内）
├ README.md              # リポジトリ概要
└ 整理/                  # 統合整理ファイル（全設定の正典）
└ 時系列/                # Y年基準の時系列
└ 人物/                  # 人物設定
└ 舞台/                  # 地理・舞台設定
└ 展開設計/              # ファイル別展開設計
└ 哲学・文体/            # 描写論・文体原則
└ 担当指示書/            # 各担当チャット向け指示書
```

---

## 各担当への共通指示

1. **まず `担当指示書/` 内の自分の担当ファイルを読む**
2. 参照先ファイルをfetchして最新情報を確認する
3. 作者の質問・依頼に専門的な立場で応じる
4. 設定の**一貫性を保つ**（矛盾があれば必ず指摘する）
5. **未確定の箇所は推測で断定しない**（`整理/v8_現時点.md` §22 参照）
6. 設定変更・追加は提案として示し、作者の判断を仰ぐ

---

## 正典ファイル

`整理/v8_現時点.md`（または最新版）が全情報の源泉。
他ファイルはここから抽出・整理されたもので、矛盾がある場合は整理ファイルを優先する。

## 作品の根本（必読）

**キャッチフレーズ**: 「人生は、旅とその痕によって出来ている。」

**本質**: 劇的な出来事ではなく、ある一日・ある瞬間・ある接触の積み重なりとして、生が形成されていく様を記録する作品。描写とは痕を残す行為。

**哲学的根底**: 存在することの不完結性への応答。矛盾するものを止揚でも妥協でもなく「包括」する倫理。

**形式の倫理**: 完結させないことへの誠実さ。整理しきれないものを、それでも手渡そうとする行為としての物語。

**枠組み**: 2080年代。夕の晩年に、澪と灯が夕凪の記録・メモ・語りを整理している。読者が読んでいるのはその「ファイル」。
