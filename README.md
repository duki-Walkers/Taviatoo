# 「旅痕」制作データベース

道路歩行 作の長編小説「旅痕」の制作委員会共有データベース。

カクヨム公開作品: https://kakuyomu.jp/works/2912051599426575918

---

## このリポジトリについて

「旅痕」の本文・設定・プロンプト・成果物の置き場。
各担当チャット（Claude Code セッション）が参照する共有データベースとして機能する。
作者が更新・コミットし、各担当チャットがfetchして最新情報を参照する運用。

## 作品概要

**キャッチフレーズ**: 「人生は、旅とその痕によって出来ている。」

**枠組み**: 2080年代。夕の晩年に、澪と灯が夕凪の記録・メモ・語りを整理している。読者が読んでいるのはその「ファイル」。

**全体構成**:
- プロローグ「遭遇」
- 第一ファイル「起源」
- 第二ファイル「出会う」
- 第三ファイル「世界」
- 第四ファイル「浮遊」
- 第五ファイル「加入」
- 第六ファイル「澪灯」
- エピローグ「夕凪」
