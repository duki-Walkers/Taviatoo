## 作品概要

**キャッチフレーズ**
「人生は、旅とその痕によって出来ている。」

**本質**
劇的な出来事ではなく、ある一日・ある瞬間・ある接触の積み重なりとして、生が形成されていく様を記録する作品。描写とは痕を残す行為。

**哲学的根底**
存在することの不完結性への応答。矛盾するものを止揚でも妥協でもなく「包括」する倫理。

**形式の倫理**
完結させないことへの誠実さ。整理しきれないものを、それでも手渡そうとする行為としての物語。

**枠組み**
2080年代。夕の晩年に、澪と灯が夕凪の記録・メモ・語りを整理している。読者が読んでいるのはその「ファイル」。

## リポジトリの使い方

### このリポジトリの役割

「旅痕」制作委員会の各担当チャット（Claude Code セッション）が参照する**共有データベース**。

- 各担当チャットがファイルをfetchして最新設定を確認する
- 作者が設定を更新・コミット・プッシュすることで全チャットへ反映される
- 担当チャットは自分の守備範囲に関する質問・依頼に専門的に応じる

### 担当チャットの起動手順

1. `CLAUDE.md` が自動的に読み込まれる（Claude Code の動作）
2. 自分の担当指示書（`担当指示書/〇〇担当.md`）を読む
3. 指示書に記載の参照ファイルをfetchして最新情報を確認する
4. 作者の質問・依頼に応じる

### 参照ファイルのfetch方法

各担当指示書に記載の GitHub URL（Raw）からファイルを取得する。

```
https://raw.githubusercontent.com/duki-Walkers/Taviatoo/main/整理/v8_現時点.md
```

### 更新フロー

```
作者が設定変更・追記
　↓
ローカルで編集 → git commit → git push
　↓
各担当チャットが次回セッション開始時にfetchして反映
```

### 正典ファイルについて

`整理/v8_現時点.md`（または最新版）が**全情報の源泉**。
各ディレクトリのファイルはここから抽出・整理されたもので、矛盾がある場合は整理ファイルを優先する。
